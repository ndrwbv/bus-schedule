## Контекст

**Зачем:** Дополнительный канал монетизации. Пользователи которые ценят сервис могут поддержать автора. Не заменяет рекламу, но создаёт лояльность и покрывает часть расходов (хостинг ~500₽/мес, домен, время разработки).

**Что есть сейчас:** Ничего. Нет кнопки донатов.

**Аудитория:** ~2K уникальных пользователей/месяц, жители Северного парка и окрестностей Томска. Люди пользуются сервисом ежедневно → высокая лояльность → потенциал для донатов.

### Детальный разбор платформ

### 1. СБП / перевод на карту Т-Банк ✅ ОСНОВНОЙ

**Суть:** Пользователь сканирует QR-код или переводит по номеру телефона через приложение своего банка.

| Параметр | Значение |
|----------|----------|
| Комиссия для автора | **0%** |
| Комиссия для донатера | **0%** (в рамках лимитов СБП) |
| Регистрация донатера | Не нужна (только приложение банка) |
| Мин. сумма | 1₽ |
| Способы оплаты | СБП (100+ банков), прямой перевод |
| Виджет | QR-код (генерируется в приложении) |
| Вывод | Мгновенно, деньги сразу на карте |
| Лимиты | До 1 000 000₽ за перевод, 5 000 000₽/мес |
| Регистрация автора | Уже есть карта → 0 действий |

**Плюсы:**
- Нулевая комиссия для обеих сторон
- Все пользователи умеют переводить по СБП
- Мгновенное зачисление
- Не нужна регистрация нигде

**Минусы:**
- Нет статистики (кто, сколько, когда)
- Нельзя автоматизировать «спасибо»
- Нужно публиковать номер телефона или QR
- Не выглядит «профессионально»

---

### 2. Donatty (donatty.com) ✅ РЕКОМЕНДУЕМЫЙ

**Суть:** Сервис для приёма донатов с самой низкой комиссией на рынке (3.5%). Платежи через НКО «МОНЕТА» (лицензия ЦБ РФ). Донаты не облагаются налогом.

| Параметр | Значение |
|----------|----------|
| Комиссия | **3.5%** — самая низкая |
| Регистрация донатера | **Не нужна** |
| Мин. сумма | 1₽ |
| Способы оплаты | Карта, ЮMoney, USDT, FK Wallet |
| Виджет | Уведомления, прогресс-бар, цель сбора, виджеты для OBS/XSplit |
| Вывод | **5–10 минут** на карту. 0₽ при выводе > 10 000₽, 30₽ при меньшей сумме. Мин. 500₽ |
| Лимиты | Макс. 60 000₽ на счёте, 200 000₽/мес |
| Налоги | Донаты **не облагаются** НДФЛ |
| Регистрация автора | Простая, верификация не требуется сразу |

**Плюсы:**
- Самая низкая комиссия (3.5% vs 5–11.7% у конкурентов)
- Донатер не регистрируется
- Быстрый вывод (5–10 мин)
- Лицензированный платёжный оператор (НКО МОНЕТА, ЦБ РФ)
- Виджеты и страница сбора с целью
- Не облагается налогом

**Минусы:**
- Менее известен чем CloudTips/Boosty
- Лимит 60 000₽ на счёте (но для нашего проекта более чем достаточно)
- Мин. сумма для вывода — 500₽

### 3. CloudTips (cloudtips.ru) ✅ АЛЬТЕРНАТИВА

**Суть:** Сервис от Т-Банка/CloudPayments для приёма чаевых и донатов. Работает через QR-код и ссылку, без регистрации для отправителя.

| Параметр | Значение |
|----------|----------|
| Комиссия | **5%** (вывод на карту Т-Банк) / **7%** (другие банки) |
| Регистрация донатера | **Не нужна** |
| Мин. сумма | 1₽ |
| Способы оплаты | Карта, СБП, T-Pay, Яндекс Pay |
| Виджет | QR-код + ссылка + виджет для сайта |
| Вывод | **Мгновенно** на привязанную карту |
| Налоги | Чаевые **не облагаются** НДФЛ |
| Регистрация автора | Простая, через приложение CloudTips |

**Плюсы:**
- Донатер не регистрируется — просто платит
- Компенсация комиссии: донатер может сам оплатить комиссию (опция в форме)
- Красивая страница с целью сбора
- Статистика донатов в ЛК
- Виджеты для OBS/сайта
- Не облагается налогом
- Банковский сервис — надёжность

**Минусы:**
- Комиссия 5–7%
- Не так известен как Boosty среди IT-аудитории

### 4. ЮMoney (yoomoney.ru) ⚠️ ЗАПАСНОЙ

**Суть:** Кошелёк ЮMoney с кнопкой/формой сбора денег. Есть API для встраивания формы на сайт.

| Параметр | Значение |
|----------|----------|
| Комиссия при приёме | **0%** (кошелёк→кошелёк в приложении) / **0.5%** на сайте |
| Комиссия при выводе | **3% + 45₽** на карту / **0%** через СБП (до 3 500₽/сутки) |
| Регистрация донатера | Не обязательна (можно с карты без кошелька) |
| Мин. сумма | 1₽ |
| Способы оплаты | Кошелёк ЮMoney, карта, СБП |
| Виджет | **Да** — кнопка, форма, API |
| Вывод | На карту или через СБП |
| Лимиты кошелька | Именной: 15 000₽ хранение / Идентифицированный: 600 000₽/мес |
| Регистрация автора | Нужен именной или идентифицированный кошелёк |

**Плюсы:**
- Кнопка для сайта через конструктор (без кода)
- API для кастомной формы
- Знакомый бренд (бывшие Яндекс.Деньги)
- Бесплатный вывод через СБП (до лимита)

**Минусы:**
- Комиссия за вывод на карту ощутимая (3% + 45₽)
- Нужна идентификация для нормальных лимитов
- Лимит бесплатного СБП-вывода — 3 500₽/сутки
- Интерфейс формы выглядит устаревшим

### 5. Boosty (boosty.to) ⚠️ ДЛЯ ПОДПИСОК

**Суть:** Платформа для создателей контента с подписками и разовыми донатами.

| Параметр | Значение |
|----------|----------|
| Комиссия | **11.7%** (с февраля 2026) |
| Регистрация донатера | **Да, обязательна** |
| Способы оплаты | Банковские карты |
| Виджет | Ссылка на страницу / QR-код |
| Вывод | Ежедневно, мин. 150₽ |
| KYC автора | Обязательна (3 уровня) |
| Налоги | Доход облагается налогом (кроме донатов) |

**Плюсы:**
- Страница проекта с целями и подписками
- Можно делать платный контент
- Статистика подписчиков
- Донаты не облагаются налогом

**Минусы:**
- **11.7% комиссия** — самая высокая
- Донатер **обязан регистрироваться** — барьер
- Оплата из РФ стала проблемной (нужны обходные пути)
- Платформа может в любой момент поднять комиссию (уже поднимали)

### 6. VK Donut ❌ НЕ ПОДХОДИТ

| Параметр | Значение |
|----------|----------|
| Комиссия | **10%** + 3% вывод на карту |
| Разовый донат | 10–15 000₽ |
| Подписка | 50–10 000₽/мес |
| Вывод | На карту до 24ч / СБП бесплатно |

**Почему не подходит:** Работает только внутри ВКонтакте. Нужно сообщество VK. Нет виджета для внешнего сайта. Для нашего проекта (SPA) — лишний слой.

---

### 7. DonationAlerts ❌ НЕ ПОДХОДИТ

| Параметр | Значение |
|----------|----------|
| Комиссия | **10%** (7% от 1M₽/мес) |
| Вывод | Проблемы с российскими картами |

**Почему не подходит:** Сервис ориентирован на стримеров. Серьёзные проблемы с выводом на российские карты (блокировки, задержки). Оплата от российских пользователей ограничена. Принадлежит VK, но хуже чем VK Donut для наших целей.

---

### 8–10. Patreon / Buy Me a Coffee / Ko-fi ❌ НЕ РАБОТАЮТ В РФ

Все три платформы фактически недоступны для российских авторов и донатеров из-за санкций:
- Patreon заблокировал российские аккаунты
- Buy Me a Coffee не принимает российские карты
- Ko-fi работает через Stripe/PayPal — оба недоступны в РФ

## UX-стратегия подачи

### Принципы
- **Сначала польза, потом просьба** — не показывать новичкам
- **Один раз сказал — отстал** — закрыл = больше не лезет (долго)
- **Честность > маркетинг** — простой текст от человека, не корпоративщина

### Точка 1: Страница «О проекте» — постоянная, ненавязчивая

Ссылка в bottom sheet или меню: **«О проекте»**. На странице:

> **severbus.ru**
>
> Привет! Меня зовут Андрей, я сделал этот сайт потому что сам жду 112С на остановке и устал смотреть расписание в таблицах.
>
> Проект поддерживается одним человеком в свободное время. Если сервис полезен — скинь на кофе, будет мотивация делать дальше ☕
>
> [Кнопка: Поддержать проект]

**Почему работает:** кто хочет — найдёт сам. Нулевая навязчивость.

### Точка 2: Мягкий баннер после 5-го визита — одноразовый

После **5-го визита** (пользователь уже лояльный) показать тонкий баннер внизу экрана:

> Этот сервис поддерживает один человек из Томска. Нравится? [Поддержать ☕] [✕]

Правила:
- Нажал **✕** → не показываем **90 дней** (localStorage)
- Нажал **«Поддержать»** → попап → больше не показываем
- Формат: тонкая полоска ~48px внизу, **не модалка**, не перекрывает контент
- Считаем визиты по дням (не по загрузкам страницы)

**Почему работает:** пользователь уже получил ценность 5 раз. Просьба уместна. Одноразовость = уважение.

### Чего НЕ делать
- ~~Попап при первом входе~~ — человек ещё не знает что это за сайт
- ~~Блокирующая модалка~~ — люди торопятся на автобус
- ~~Счётчик «собрано X из Y»~~ — при 2K пользователях цифры будут грустные
- ~~Регулярные напоминания~~ — закрыл = закрыл

## User Stories

### US-1 — Страница «О проекте» с кнопкой доната (Приоритет: P1)

Постоянная ссылка в bottom sheet. Минимум работы, сразу даёт возможность поддержать.

**Сценарии приёмки:**

1. **Дано** пользователь на сайте, **Когда** нажимает «О проекте» в bottom sheet, **Тогда** открывается страница с текстом и кнопкой доната
2. **Дано** пользователь нажал «Поддержать проект», **Тогда** открывается попап с номером телефона, ссылкой на Т-Банк и текстом
3. **Дано** мобильное устройство, **Когда** нажимает «Перевести», **Тогда** открывается ссылка Т-Банка для перевода
4. **Дано** десктоп, **Тогда** видно номер телефона для ручного перевода

### US-2 — Мягкий баннер после 5-го визита (Приоритет: P2)

Тонкий баннер внизу экрана, показывается один раз.

**Сценарии приёмки:**

1. **Дано** пользователь зашёл 5-й раз (по дням), **Тогда** показывается баннер внизу
2. **Дано** баннер показан, **Когда** нажимает ✕, **Тогда** баннер исчезает и не показывается 90 дней
3. **Дано** баннер показан, **Когда** нажимает «Поддержать», **Тогда** открывается попап доната и баннер больше не показывается
4. **Дано** пользователь зашёл 1–4 раз, **Тогда** баннер НЕ показывается

## Edge Cases

- Кнопка «О проекте» не должна быть навязчивой — в bottom sheet, маленькая, не акцентная
- Баннер не перекрывает расписание и кнопки навигации
- Счётчик визитов: считать уникальные дни в localStorage, не загрузки страницы
- Если localStorage очищен — счётчик сбрасывается, баннер может показаться снова (это ок)
- На мобильном ссылка Т-Банка открывает приложение банка (если установлено) или страницу перевода

## Критерии приёмки

- [ ] Ссылка «О проекте» в bottom sheet
- [ ] Страница «О проекте» с текстом и кнопкой «Поддержать проект»
- [ ] Попап с номером телефона Т-Банк и ссылкой на перевод
- [ ] Баннер после 5-го визита (по дням) с кнопками «Поддержать» и «✕»
- [ ] Баннер скрывается на 90 дней после закрытия (localStorage)
- [ ] Адаптивный дизайн попапа и баннера (мобильный / десктоп)
