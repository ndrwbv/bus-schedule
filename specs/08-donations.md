# Фича: Донаты

**Ветка:** `08-donations`
**Создана:** 2026-03-25
**Статус:** черновик

## Контекст

**Зачем:** Дополнительный канал монетизации. Пользователи которые ценят сервис могут поддержать автора. Не заменяет рекламу, но создаёт лояльность и покрывает часть расходов.

**Что есть сейчас:** Ничего. Нет кнопки донатов.

## Варианты реализации

### Вариант А: Перевод на карту Тинькофф

| Плюсы | Минусы |
|-------|--------|
| Ноль настройки | Нет статистики кто и сколько перевёл |
| Все знают как переводить | Нельзя автоматизировать «спасибо» |
| Нет комиссии платформы | Нужно публиковать номер карты/телефона |
| Мгновенное поступление | Не выглядит «профессионально» |

### Вариант Б: Boosty / ЮMoney

| Плюсы | Минусы |
|-------|--------|
| Статистика, подписчики | Комиссия 10-15% |
| Выглядит как настоящий проект | Ещё одна платформа для пользователя |
| Можно сделать «цели» (на хостинг, на разработку) | Нужна регистрация для донатера |
| Виджет/кнопка для встраивания | |

### Вариант В: Комбинированный (рекомендуемый)

- Кнопка «Поддержать проект» на сайте
- Открывает попап с двумя вариантами:
  1. «Перевести на карту» — QR-код / номер телефона Тинькофф (быстро, без регистрации)
  2. «Поддержать на Boosty» — ссылка (для тех кто хочет подписку)
- Минимальный текст: «Проект делает один человек для района. Любая сумма помогает.»

## User Stories

### US-1 — Кнопка «Поддержать проект» (Приоритет: P1)

На сайте появляется кнопка, открывающая попап с вариантами доната.

**Почему этот приоритет:** Минимум работы, потенциально ненулевой доход с первого дня.

**Сценарии приёмки:**

1. **Дано** пользователь на сайте, **Когда** нажимает «Поддержать проект», **Тогда** открывается попап с вариантами перевода
2. **Дано** попап открыт, **Тогда** видно: QR-код для перевода + ссылка на Boosty + короткий текст
3. **Дано** мобильное устройство, **Тогда** попап адаптивный, QR-код достаточно крупный

---

## Edge Cases

- Кнопка не должна быть навязчивой — показывать в футере bottom sheet, не всплывать автоматически
- QR-код Тинькофф: проверить что работает для перевода по номеру телефона через Тинькофф QR
- Если Boosty закроется? → Кнопка Boosty просто убирается, перевод на карту остаётся

## Критерии приёмки

- [ ] Кнопка «Поддержать проект» в bottom sheet
- [ ] Попап с QR-кодом и ссылкой на Boosty
- [ ] Текст объясняющий зачем донаты
- [ ] Адаптивный дизайн попапа
