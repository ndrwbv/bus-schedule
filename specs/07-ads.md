# Фича: Реклама

**Ветка:** `07-ads`
**Создана:** 2026-03-25
**Статус:** черновик

## Контекст

**Зачем:** Монетизация — чтобы проект окупал хостинг и мотивировал поддерживать. Два канала: автоматическая реклама (РСЯ) и ручная локальная реклама (баннеры местных бизнесов).

**Что есть сейчас:** Компонент `src/features/StripAd/` — уже есть место под рекламный баннер на фронте. Ранее использовался для кастомной рекламы (курсы по танцам — нерелевантно, не кликали).

## User Stories

### US-1 — РСЯ (Рекламная Сеть Яндекса) (Приоритет: P1)

Подключить РСЯ — автоматически подбирает релевантную рекламу. Один блок в bottom sheet (между расписанием и жалобами).

**Почему этот приоритет:** Минимум усилий, максимум автоматизации. Не надо искать рекламодателей.

**Как проверить независимо:** Открыть сайт → увидеть рекламный блок РСЯ.

**Сценарии приёмки:**

1. **Дано** пользователь открыл сайт, **Когда** скроллит bottom sheet, **Тогда** видит рекламный блок РСЯ
2. **Дано** РСЯ не загрузилась (adblock или ошибка), **Тогда** блок не показывается, layout не ломается
3. **Дано** мобильное устройство, **Тогда** размер баннера адаптивный

---

### US-2 — Локальные баннеры (Приоритет: P2)

Возможность показывать баннер местного бизнеса вместо (или рядом с) РСЯ. Баннер управляется через конфиг на бэкенде (URL картинки, ссылка, срок показа).

**Почему этот приоритет:** Потенциально больше денег чем РСЯ, но требует ручной работы по поиску рекламодателей.

**Сценарии приёмки:**

1. **Дано** настроен локальный баннер, **Когда** пользователь открывает сайт, **Тогда** видит баннер местного бизнеса
2. **Дано** срок показа баннера истёк, **Тогда** показывается РСЯ (fallback)
3. **Дано** нет ни локального баннера, ни РСЯ, **Тогда** блок скрыт

---

## Edge Cases

- Что если РСЯ отклонит заявку на подключение? → Альтернативы: Google AdSense, MyTarget
- Что если рекламу режет adblock? → Ничего не делать, не ломать UX
- Где именно размещать баннер? → В bottom sheet между ключевым контентом, не на карте (мешает UX)

## Критерии приёмки

- [ ] РСЯ баннер отображается в bottom sheet
- [ ] Graceful degradation: если реклама не загрузилась — блок скрыт
- [ ] [P2] Локальный баннер с управлением через бэкенд
- [ ] [P2] Fallback: локальный баннер → РСЯ → скрыть
