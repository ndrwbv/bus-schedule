# Фича: Привлечение локальных рекламодателей

**Ветка:** —
**Создана:** 2026-03-25
**Статус:** черновик

## Контекст

**Зачем:** Локальная реклама — потенциально самый доходный канал (2000+ руб/мес за баннер). Нужно системно подойти к поиску рекламодателей: подготовить оффер, список бизнесов, шаблон письма/сообщения.

**Что есть сейчас:** Ничего — раньше была только нерелевантная реклама (курсы по танцам).

## План действий

### 1. Подготовить медиакит (когда сайт заработает)

Одностраничный PDF или страница на сайте с:
- Что такое severbus.ru (районный сервис расписания автобусов)
- Аудитория: ~2000 жителей Северного парка в месяц
- Формат рекламы: баннер в приложении (скриншот с примером размещения)
- Стоимость: 2000 руб/мес (стартовая цена, можно гибко)
- Контакт для связи

### 2. Список потенциальных рекламодателей

Бизнесы рядом с остановками Северного парка:

- [ ] Пекарни / кофейни
- [ ] Аптеки
- [ ] Продуктовые магазины (не сети — локальные)
- [ ] Доставка еды (локальная)
- [ ] Автосервисы
- [ ] Стоматологии / клиники
- [ ] Салоны красоты
- [ ] Детские центры
- [ ] Автошколы
- [ ] Риелторы (новостройки в Северном парке)

### 3. Повод для обращения

**Основной посыл:** «Поддержка районного проекта»

Формулировка для первого контакта:

> Здравствуйте! Я разработчик сайта severbus.ru — это районный сервис для жителей Северного парка, который показывает расписание автобусов в удобном формате. Сайтом пользуются ~2000 человек в месяц — это ваши соседи и потенциальные клиенты.
>
> Ищу локальных партнёров для размещения рекламы. Баннер вашего бизнеса будут видеть жители района каждый день, пока ждут автобус. Стоимость — 2000 руб/мес.
>
> Буду рад обсудить детали. Могу показать сайт и статистику.

**Альтернативные поводы:**
- «Первый месяц бесплатно — попробуйте и оцените результат»
- «Поддержите локального разработчика — ваш баннер увидят 2000 жителей района»
- Через группу VK: «Ищу партнёра для размещения рекламы — может кто знает бизнесы которым это интересно?»

### 4. Когда начинать

**Не раньше чем:**
- [ ] Сайт работает с актуальным расписанием
- [ ] Есть статистика посещений за 2+ недели (из своего мониторинга)
- [ ] Есть скриншот с примером размещения баннера

**Порядок:**
1. Написать 2-3 ближайшим бизнесам (пекарня, аптека)
2. Если откажут — предложить бесплатный месяц
3. Если согласятся — первый кейс для привлечения следующих

## Критерии приёмки

- [ ] Медиакит готов (PDF или страница)
- [ ] Список 10+ потенциальных рекламодателей с контактами
- [ ] Шаблон первого сообщения
- [ ] Первые 3 обращения отправлены
