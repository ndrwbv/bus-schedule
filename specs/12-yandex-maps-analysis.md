# Анализ: Миграция на Яндекс Карты

**Создана:** 2025-04-11
**Статус:** исследование

## Контекст

**Зачем:** Оценить целесообразность миграции с MapLibre GL + OpenFreeMap на Яндекс Карты JavaScript API 3.0. Яндекс Карты привычнее для российских пользователей и имеют лучшую детализацию в Томске.

**Что есть сейчас:** MapLibre GL `4.7.1` + тайлы OpenFreeMap (`tiles.openfreemap.org/styles/liberty`). Leaflet присутствует в зависимостях, но не используется. MapTiler ключи в секретах, но тоже не используются.

## 2. Используемые фичи MapLibre и аналоги в Яндекс Картах 3.0

| Фича MapLibre | Яндекс API 3.0 аналог | Примечания |
|---|---|---|
| `new maplibregl.Map({...})` | `new YMap(el, {location: {...}})` | Другой API инициализации |
| `map.addSource('id', {type: 'geojson'})` | `YMapFeatureDataSource` | Принципиально другая модель — нет source/layer, вместо этого компонентная модель |
| `map.addLayer({type: 'circle'})` | `YMapMarker` с кастомным DOM | Нет аналога circle layer — нужно через DOM-маркеры |
| `map.addLayer({type: 'symbol'})` | `YMapMarker` с кастомным DOM | Нет аналога symbol layer — нужно через DOM-маркеры |
| GeoJSON clustering (Supercluster) | `@yandex/ymaps3-clusterer` | Есть пакет, но API совершенно другой |
| `maplibregl.Marker({element})` | `YMapMarker({element})` | **Похоже** — оба принимают DOM-элемент |
| `map.flyTo({center, zoom})` | `map.setLocation({center, zoom, duration})` | Простая замена |
| `map.easeTo({center, zoom})` | `map.setLocation({center, zoom, duration})` | Простая замена |
| `map.queryRenderedFeatures(point)` | Нет прямого аналога | Нужно вручную обрабатывать клики по маркерам |
| `map.querySourceFeatures(id)` | Нет прямого аналога | Нужно хранить состояние маркеров вручную |
| `map.addImage() / updateImage()` | DOM-маркер с `<canvas>` или `<img>` | Нет API для кастомных изображений в слоях |
| `map.setPaintProperty()` | Прямая манипуляция DOM маркера | Нет аналога — анимации через DOM |
| `map.getZoom()` | `map.zoom` | Простая замена |
| `map.on('zoomend'/'dragend')` | `YMapListener({onUpdate})` | Другая модель событий |
| `map.on('click', layerId, cb)` | Обработчик на DOM-элементе маркера | Нет layer-level событий |
| `map.getCanvas().style.cursor` | CSS на контейнере карты | Простая замена |
| `pitch: 45, bearing: 60` | `tilt, azimuth` | Поддерживается |
| `attributionControl: false` | `YMapControls` — можно не добавлять | Простая замена |

### Критическое различие

MapLibre использует **data-driven модель** (sources → layers → paint/layout properties). Яндекс Карты 3.0 используют **компонентную модель** (React-подобные YMap* компоненты). Это означает, что вся логика слоёв (clusters circle layer, unclustered-point circle layer, pulse circle layer, symbol layers) должна быть **полностью переписана** на DOM-маркеры с CSS-анимациями.

## 3. Стоимость Яндекс Карт

### Бесплатный тариф (подходит для severbus.ru)

| Параметр | Лимит |
|---|---|
| Загрузка карты (тайлы) | **Без ограничений** |
| Геокодер (поиск по адресу) | 1 000 запросов/сутки |
| Суммарно тарифицируемых запросов | 25 000/сутки |
| Маршрутизатор | В пределах 25 000/сутки |

**Маркеры по координатам не тарифицируются.** При ~2K уников/месяц проект далеко от лимитов.

### Условия бесплатного использования
- Сайт должен быть **публичным** (severbus.ru — подходит)
- Данные должны отображаться **на карте Яндекса** (подходит)
- **Запрещено** использовать для мониторинга/диспетчеризации транспорта
  - Live tracking (показ автобуса на карте) **может попасть под это ограничение** — требует уточнения у Яндекса
- Нельзя сохранять или модифицировать полученные данные (мы не делаем этого)

### Платные тарифы (для справки)
- От 16 000 руб/мес (JavaScript API + Геокодер)
- От 150 000 руб/год
- Перерасход: 360 руб за 1 000 запросов

**Вывод по цене: бесплатно** для текущего объёма, но live tracking — зона риска.

## 4. Оценка сложности миграции

### Простые замены (1-2 часа)
- `mapProvider.ts` — удалить, подключить Яндекс API скрипт
- `TMap.ts` — заменить тип
- `Map.tsx` — новая инициализация через `YMap`
- `MyLocation` — `flyTo` → `setLocation`
- Удалить неиспользуемые `leaflet`, `@types/leaflet`

### Средние переделки (4-6 часов)
- `MapContent.tsx` — маркеры остановок: DOM-маркеры вместо GeoJSON source + `querySourceFeatures`. Сам пин (SVG + HTML) переиспользуется
- Кластеризация — подключить `@yandex/ymaps3-clusterer`, переписать конфиг
- События карты — `YMapListener` вместо `map.on()`

### Тяжёлые переделки (8-12 часов)
- `LiveBusLayer.tsx` — нет circle layer и symbol layer, нужно:
  - DOM-маркер с CSS-анимацией пульса (вместо `circle-radius` + `circle-opacity` через `setPaintProperty`)
  - RAF-анимация позиции через `marker.update({coordinates})` вместо GeoJSON source update
  - Загрузка SVG иконки через DOM (вместо `map.addImage`)
- `MapAdBanner.tsx` — нет `addImage`/`updateImage`, нужно:
  - DOM-маркер с `<canvas>` элементом внутри
  - Анимация fade через CSS opacity вместо перерисовки image
  - Клик через DOM event вместо `queryRenderedFeatures`
- Скрытие текстовых лейблов карты — в Яндексе через кастомизацию стиля карты (поддерживается в 3.0)

### Итого: ~15-20 часов работы

## 5. Риски

| Риск | Серьёзность | Митигация |
|---|---|---|
| Live tracking может нарушать условия бесплатного использования | **Высокая** | Уточнить у Яндекса: показ 1-2 автобусов на расписании ≠ диспетчеризация |
| Яндекс API 3.0 документация неполная | Средняя | Много примеров для 2.1, community-ресурсы |
| Производительность DOM-маркеров vs WebGL-слоёв | Низкая | У нас <50 маркеров, DOM справится |
| Vendor lock-in на Яндекс | Средняя | OpenFreeMap/MapLibre останется как fallback |
| Яндекс может изменить условия/цены | Средняя | Сохранить MapLibre код в отдельной ветке |

## 6. Альтернативы

| Вариант | Цена | Сложность миграции | Детализация в Томске |
|---|---|---|---|
| **Оставить OpenFreeMap + MapLibre** | Бесплатно | 0 | Средняя (OSM данные) |
| **Яндекс Карты 3.0** | Бесплатно* | ~15-20ч | Отличная |
| **2GIS API** | Бесплатно (до 10К/мес) | ~15-20ч | Отличная (особенно для Томска) |
| **MapTiler + MapLibre** | $0-100/мес | ~1ч (только URL тайлов) | Средняя |
| **Google Maps** | $200 кредит/мес | ~20ч | Хорошая |

## 7. Рекомендация

**Если основная цель — лучшая карта для пользователей в Томске:**
- 2GIS может быть даже лучше Яндекса для городского транспорта
- Яндекс — хороший выбор, знакомый всем пользователям

**Если основная цель — минимум затрат:**
- Оставить OpenFreeMap + MapLibre (текущее решение) — 0 работы, 0 рублей
- Или просто сменить URL тайлов на MapTiler (ключи уже есть в секретах)

**Если решили мигрировать на Яндекс:**
1. Уточнить у Яндекса про live tracking
2. Начать с простой карты + маркеры (без live bus и ad banner)
3. Добавить кластеризацию
4. Портировать LiveBusLayer на DOM-маркеры
5. Портировать MapAdBanner на DOM-маркер с canvas
